# Blink-it-Dashboard

I have built this interactive dashboard to understand sales performance across BlinkIT outlets in a more clear and practical way.

The dashboard allows multi-dimensional analysis of sales, items, and outlet data - making it easier to identify trends, compare performance, and understand what’s actually driving revenue.

KPI Cards :

1. Total Sales : Overall revenue generated for the selected data
2. Average Sales : Average revenue per transaction
3. Number of Items : Total items available/sold
4. Average Rating : Average customer rating across products

Visual Components : 

1. Outlet Establishment Trend : Line chart showing how sales vary based on outlet establishment over time

2. Sales by Item Type : Horizontal bar chart highlighting top-performing categories like fruits, snacks, dairy, etc.

3. Fat Content Analysis : Donut chart showing contribution of low fat vs regular items

4. Fat by Outlet : Bar chart comparing fat category sales across Tier 1, Tier 2, and Tier 3 outlets

5. Outlet Size Distribution : Donut chart representing sales contribution by small, medium, and high outlets

6. Sales by Outlet Location : Bar chart comparing Tier 1, Tier 2, and Tier 3 performance

7. Outlet Type Table : Detailed breakdown of supermarket types and grocery stores with sales, items, ratings, and visibility

Key Insights :

1. Tier 3 outlets drive the highest sales
2. Regular items sell more than low-fat ones
3. Fruits, snacks, and household items lead revenue
4. Sales vary clearly across outlet types
